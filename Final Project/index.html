Lab Reflections

Lab 01 - Web Publishing

This lab introduced me to the world of web publishing. I learned how to host a simple web page using GitHub Pages, which was the first step in showcasing my work online. It was exciting to see my webpage live and accessible to others for the first time.

Lab 02 - HTML Basics

Lab 02 helped me understand the structure of HTML, including tags, elements, and how to organize content on a web page. I now feel confident in writing basic HTML code, which is foundational for building any web page.

Lab 03 - CSS Basics

In this lab, I learned the basics of CSS and how to apply styles to HTML elements. The ability to change colors, fonts, and layouts significantly improved the look and feel of my website. It was an essential step in creating visually appealing pages.

Lab 04 - Visual Media Lab

Lab 04 focused on integrating images into a web page. I learned about different image formats, how to choose the best one for various scenarios, and how to properly style and position them using CSS. This lab greatly enhanced my ability to work with visual media online.

Lab 05 - Layout Lab

Lab 05 introduced me to more advanced CSS techniques like Flexbox and Grid. I gained hands-on experience building responsive layouts, which is crucial for modern web design. I now feel more capable in creating flexible and organized layouts that work across devices.

Lab 06 - UI/UX Evaluation

In this lab, I learned how to evaluate the usability of a website from both a user interface (UI) and user experience (UX) perspective. This knowledge has helped me develop a deeper understanding of how design affects the user's interaction with a website.

Lab 07 - Tables & Forms

Lab 07 focused on creating tables and forms, which are key components for many types of websites. I learned how to structure data using tables and collect user input through forms, both of which are critical for dynamic, interactive sites.

Lab 08 - Interactive Menu

Lab 08 taught me how to add interactive elements to a website, such as a navigation menu. I used JavaScript to create an engaging user experience, allowing visitors to navigate my site more easily.

Lab 09 - XML

Lab 09 introduced me to XML, a markup language used for storing and transporting data. I learned how to create and modify XML files, which is an essential skill for working with structured data in web development.

Lab 10 - Web Standards

In Lab 10, I researched and created a presentation about web standards, focusing on the importance of consistent design and accessibility. This lab helped me understand the broader context of web development and the guidelines that ensure websites are user-friendly and functional.
